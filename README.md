# FAQs

Angular FAQ application with modern architecture.

## Stack

- **Framework**: Angular 20
- **Styling**: Tailwind CSS 4 + clsx
- **Icons**: @ng-icons/lucide
- **UI**: Spartan-ng brain

## Getting Started

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Features

- FAQ list with sidebar navigation
- Search functionality
- Loading states
- Responsive design
